# VCET College Website

A modern, fully responsive, and component-based official college website for VCET, built with React and Tailwind CSS. The website features a premium, edge-to-edge design system, optimized for fast performance, accessibility, and high visual impact.

## Features
- **Responsive Layout:** Mobile-first approach using Tailwind CSS.
- **Modern UI/UX:** Built with glassmorphism, structured typography, and dynamic animations.
- **Component-Based Architecture:** Scalable and maintainable React structure.

## Tech Stack
- Frontend: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- Styling: [Tailwind CSS](https://tailwindcss.com/)
- Icons: [Lucide React](https://lucide.dev/) or other icon sets

## Running Locally

1. Clone or download the repository
2. Change directory into the project:
   ```bash
   cd vcet-website
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and navigate to `http://localhost:5173/` or the port specified in your terminal.

## Building for Production

To create a production-ready build:
```bash
npm run build
```
This will bundle the application into the `dist` folder.
